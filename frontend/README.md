# CET Predictor React Frontend

This is a modern React frontend for the CET College Predictor, built with Vite, Tailwind CSS, and Axios.

## Prerequisites
- Node.js installed
- FastAPI backend running at http://localhost:8000

## Setup Instructions

1.  **Navigate to the react_app directory**:
    ```bash
    cd react_app
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Start the Development Server**:
    ```bash
    npm run dev
    ```

4.  **Access the App**:
    Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features
- **Authentication**: Login, Registration, and a 3-step Password Recovery.
- **College Prediction**: Advanced filtering and prediction based on MHT-CET results.
- **Responsive Design**: Modern dark-themed UI styled with Tailwind CSS.
- **Smart Cards**: Results categorized into Safe Picks, Good Matches, and Stretch Colleges.
